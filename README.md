# transition-agenda
Collaboration around a public engagement toolkit for Mayor Brandon Scott's early-term agenda. 

Link to 100 Day Tracker:
https://docs.google.com/spreadsheets/d/1lkTa3zDSJ-aO0c0XxfocQP0cGPRQr68YYYwYPPBp3qo/edit?usp=sharing

The purpose of this project is to create a public web page where any Baltimore City resident can easily track the accomplishments of Mayor Scott's first term. Data will be compiled through the transition committees.
